# 📱 Nice Gadgets – Phone Catalog

This project is a modern e-commerce platform designed for browsing and purchasing electronic gadgets. It provides a smooth shopping experience with advanced filtering, product categorization, and a fully responsive interface.

The primary focus of this project was to build a scalable and type-safe application using React and TypeScript, ensuring robust performance and maintainable code.

## 🌐 Live Preview

Explore the catalog and test the shopping functionality live on GitHub Pages:
**[View Live Site](https://romanplyta.github.io/react_ts_phone-catalog/)**

## 🚀 Technology Stack

* **Framework:** `React` – Powering the UI with a component-based architecture and efficient state management.
* **Language:** `TypeScript` – Ensures type safety, reducing runtime errors and improving developer experience.
* **Routing:** `React Router v6` – Handles seamless navigation between home, catalog, and product detail pages.
* **Styling:** `SASS / SCSS` – Modular styling with variables and mixins for a consistent and professional look.
* **Build Tool:** `Vite` – Provides a lightning-fast development environment and optimized production builds.

## ✨ Key Features

* **Product Sliders:** Interactive carousels for featured products and new arrivals.
* **Advanced Filtering:** Sort and filter gadgets by category, price, and technical specifications.
* **Dynamic Catalog:** Auto-updating product lists with search functionality.
* **Shopping Cart:** Fully functional cart system to manage selected items.
* **Responsive Design:** Optimized for a flawless experience on smartphones, tablets, and desktops.

## ⚙️ Installation and Setup

To run this project locally, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/RomanPlyta/react_ts_phone-catalog.git
    ```

2.  **Navigate to the Project Directory:**
    ```bash
    cd react_ts_phone-catalog
    ```

3.  **Install Dependencies:**
    ```bash
    npm i
    ```

4.  **Launch the Project:**
    ```bash
    npm run dev
    ```
    *(This command will start the Vite development server. Open the link provided in your terminal to view the project).*
